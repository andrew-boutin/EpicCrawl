# Epic Crawl

This is a 2D top down tile based ARPG written in Java utilizing SQLite and JDBC. It also provides a level designer to the player so they can build their own adventure!

![Epic Crawl Gif](docs/epiccrawl.gif)

## Cmds

Build and run

    ./gradlew build run
